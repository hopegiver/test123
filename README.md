# 온라인 테트리스 게임

웹 브라우저에서 플레이할 수 있는 클래식 테트리스 게임입니다.

## 🎮 게임 특징

- **7가지 테트로미노 블록**: I, O, T, S, Z, J, L
- **부드러운 애니메이션**: Canvas를 활용한 부드러운 렌더링
- **점수 시스템**: 라인 클리어에 따른 점수 획득
- **레벨 시스템**: 10줄마다 레벨 증가 및 속도 증가
- **다음 블록 미리보기**: 다음에 올 블록을 미리 확인 가능
- **반응형 디자인**: 모바일과 데스크톱 모두 지원

## 🎯 조작법

- **← →**: 블록 좌우 이동
- **↓**: 빠른 낙하 (점수 +1)
- **↑**: 블록 회전
- **스페이스바**: 일시정지/재개

## 🚀 시작하기

1. 저장소를 클론하거나 다운로드합니다.
2. `index.html` 파일을 웹 브라우저에서 엽니다.
3. "게임 시작" 버튼을 클릭하여 게임을 시작합니다.

## 📁 파일 구조

```
.
├── index.html      # 게임 HTML 구조
├── style.css       # 스타일시트
├── tetris.js       # 게임 로직
└── README.md       # 프로젝트 설명
```

## 🎨 기술 스택

- HTML5
- CSS3 (그라데이션, 애니메이션)
- JavaScript (ES6+)
- Canvas API

## 📝 게임 규칙

1. 블록이 화면 아래로 떨어집니다.
2. 블록을 회전하거나 이동시켜 가로로 가득 찬 줄을 만듭니다.
3. 가득 찬 줄은 자동으로 제거되고 점수를 획득합니다.
4. 블록이 화면 상단에 쌓이면 게임 오버입니다.

## 🏆 점수 시스템

- 1줄 클리어: 100 × 레벨
- 2줄 클리어: 300 × 레벨
- 3줄 클리어: 500 × 레벨
- 4줄 클리어: 800 × 레벨

## 📄 라이선스

이 프로젝트는 자유롭게 사용 가능합니다.
